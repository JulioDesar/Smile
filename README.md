# Contributing

Clone de repository, open its folder and install dependencies with:

```sh
npm i
```

Run it using:

```sh
npm start
```

## Dependencies

- Node.js
- Axios
- React-Icons
- React Router DOM

## Features

- [x] Landing page
- [x] Dashboard admin
- [x] CRUD Client
- [x] CRUD Consultation
